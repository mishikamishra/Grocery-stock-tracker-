# Grocery Inventory Manager  
*CS Assignment – Simple Inventory System (Python)*  

### what this is  
super basic console-based grocery inventory program i made for the python project. lets you add items, update stock (restock or sell), see everything with low-stock warnings, and calculate total inventory value. works 100%, no crashes.

### features  
- add new items with price & quantity  
- update stock (+ or – works, won't go negative)  
- shows full list sorted + flags anything under 10 units  
- calculates total $ value of everything in stock  
- simple menu, no weird libraries needed  

### how to run  
1. make sure you have python 3 installed (literally any version works)  
2. save the code as `inventory.py` (or whatever name you want)  
3. open terminal / command prompt  
4. run:  
   ```bash
   python inventory.py
